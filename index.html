File Handling
File Handling is a way to store data on your computer.

With Python, we can:
  • Write data to a file
  • Read data from a file
Opening a File
  • open(): Creates a link between your program and the file on your system.
  • Purpose: Tells Python which file to work with and in what mode (read, write, or append).
  • Requirement: Without open(), Python won't know the file's location or purpose.
Syntax:
file = open("filename.txt", "mode")
  • "filename.txt": The name of the file.
  • "mode": Tells Python what you want to do with the file(read,write, append). The default mode is read mode(‘r’)
File Modes
While opening the file, we can specify the mode in which we want to interact with the file like reading, writing, or appending.

Different Modes:
  • Read Mode ('r'): Opens the file for reading only.
  • Write Mode ('w'): Opens the file for writing.It will overwrite existing content if the file exists and if it doesn’t it will create a new file.
  • Append Mode ('a'): Opens the file for appending which means, it allows you to add new content at the end of the file without changing the existing data.

Reading a File
  • read() reads the file’s content as a string and loads it into your program for further processing.
  • Opening a file only establishes access and the read() function extracts the data stored in the file for use in your program.
Syntax:
data = file.read()
Writing to a File
  • The write() function as the name suggests lets you add or modify the contents of a file.
Syntax:
file.write('New Data')


After executing this command you will notice the content of the file has been overwritten with the new content.

Right now the changes are taking some time to get reflected in the file, we will fix this issue in a while.

But, if you open the file in append mode(a), it retains the original content of the file.


You must have noticed that the changes we are making are not getting immediately reflected, do you know why?

This is because the changes are not being saved immediately. To avoid this we can use the close() function.
Closing a File
  • close() ensures that any changes made to the file are saved and the file is properly released for other programs.
  • If you do not close the file, the operating system may not immediately save changes, and the file might remain locked, causing issues for future access.
Syntax:
file.close()
